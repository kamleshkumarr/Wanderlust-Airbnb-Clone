# 🏡 Wanderlust – Airbnb Clone (MERN Stack)

🔗 **Live Demo:** https://wanderlust-airbnb-clone-8tjz.onrender.com/

---

## 📌 Overview

Wanderlust is a full-stack Airbnb-inspired web application that allows users to explore, list, and review rental properties. It replicates core functionalities of modern rental platforms by providing a seamless experience for both guests and hosts.

---

## 🔥 Features

### 👤 User Features

* 🔐 User Authentication (Login/Signup)
* 🏠 Browse property listings with images and details
* ⭐ Add reviews and ratings to listings
* 📱 Responsive UI for smooth user experience

---

### 🏘 Host Features

* ➕ Add new property listings
* ✏️ Edit and delete listings
* 📸 Upload property images
* 💰 Set pricing and property details

---

### ⚙️ Core Functionalities

* 📍 Location-based listing system
* 🗂 Category-wise browsing (mountains, pools, etc.)
* 🧾 Structured property details with pricing
* 🔄 Full CRUD operations for listings

---

## 🛠 Tech Stack

**Frontend:**

* HTML, CSS, JavaScript, EJS

**Backend:**

* Node.js, Express.js

**Database:**

* MongoDB (MongoDB Atlas)

**Authentication & Tools:**

* Passport.js
* Express Session
* Cloudinary (for image storage)
* Render (Deployment)
